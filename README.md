# Stock Portfolio Tracker

A simple project that calculates the investment value and total portfolio value of stocks.

## Features

- User inputs stock name and quantity
- Calculates investment value
- Calculates total portfolio value

## Technologies Used

- Python
- Input/Output
- Dictionary
- While Loop
- f-Strings
